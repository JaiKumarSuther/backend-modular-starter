# Changelog

All notable changes to this project will be documented in this file.

## 1.0.1

- Accept project name as a positional argument (e.g. `npx backend-modular-starter my-project`).
- Add `--yes/-y` and `--help/-h` flags.
- Publish metadata polish: repository/homepage/bugs, include changelog in package files.
- Generated projects use `node --watch` for `npm run dev` and set Node engines to `>=18`.

## 1.0.0

- Initial release.
